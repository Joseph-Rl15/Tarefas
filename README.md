
Um aplicativo simples de lista de tarefas feito em **Python** usando **Tkinter** e **JSON** para salvar os dados.

Projeto criado para praticar:
Interface gráfica (GUI)
Manipulação de arquivos JSON
Lógica de programação
Treeview (ttk)



Funcionalidades
Adicionar tarefas

Remover tarefas

Marcar como concluída (fica verde)  

Salvar automaticamente em JSON  

Carregar tarefas ao abrir o app
